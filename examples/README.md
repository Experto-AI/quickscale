# examples

Illustrative, opt-in reference material for QuickScale users and maintainers.

Examples in this directory are intentionally lightweight. They document patterns and experiments that can be copied selectively, but they are not generated into projects by default unless a workflow explicitly says otherwise.

## Current examples

| Example | Purpose |
|---------|---------|
| [client_extensions](./client_extensions/README.md) | Minimal pattern for project-specific backend customization without expanding the default generator output |

## Notes

- Treat examples as reference material, not authoritative product scope.
- Copy only the parts that fit a generated project's needs.
- For repository-wide policy and generated-project rules, defer to [../README.md](../README.md), [../docs/technical/decisions.md](../docs/technical/decisions.md), and [../docs/technical/scaffolding.md](../docs/technical/scaffolding.md).
